# CT É Preciso Acreditar

Site institucional da **CT É Preciso Acreditar LTDA**, comunidade terapêutica localizada em Goiana, Pernambuco.

## Objetivo do projeto

Este projeto tem como objetivo apresentar, de forma clara e profissional, as principais informações institucionais da comunidade terapêutica, incluindo:

- apresentação da instituição;
- equipe de atendimento;
- funcionamento do processo de acolhimento;
- estrutura disponível;
- canais de contato;
- informações legais e política de privacidade.

## Tecnologias utilizadas

- HTML5
- CSS3
- Git
- GitHub Pages
- Visual Studio Code

## Estrutura do site

O site possui as seguintes seções principais:

- **Hero:** chamada principal com botões de WhatsApp e telefone;
- **Sobre:** apresentação da comunidade terapêutica;
- **Equipe:** descrição dos profissionais envolvidos;
- **Como funciona:** etapas do atendimento;
- **Estrutura:** informações sobre capacidade e funcionamento;
- **Contato:** endereço, telefone, e-mail e chamada para orientação;
- **Rodapé:** dados legais, CNPJ e link para política de privacidade.
